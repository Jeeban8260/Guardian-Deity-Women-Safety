# Guardian Deity — Women Safety Wearable Intelligence System

## About
AI-powered wearable safety system that detects 
distress audio signals and automatically triggers 
GPS-tagged SOS alerts within 3 seconds.

## Tech Stack
Python | Machine Learning | TCN | IoT | GPS | 
Audio Processing (MFCC) | SOS API

## ML Models Used
- Random Forest
- Logistic Regression  
- Gaussian Naive Bayes
- Temporal Convolutional Network (TCN)

## Results
- 90%+ overall model accuracy
- 18% accuracy improvement using TCN
- SOS alert dispatch in under 3 seconds
- Trained on 1,200+ labelled audio samples

## Dataset
Publicly available at Mendeley Data:
https://data.mendeley.com/datasets/gfvsdtnf3v/1
